# Proyecto Web Moville

Este proyecto es una página web desarrollada con:
- PHP
- HTML
- JavaScript

## Funciones
- Sistema de usuarios
- Comentarios
- Panel de administrador
- Subida de imágenes

## Cómo ejecutarlo
1. Instalar XAMPP
2. Colocar el proyecto en htdocs
3. Abrir localhost


 Cómo descargar el proyecto
Opción 1 (Fácil)

Haz clic en el botón Code

Selecciona Download ZIP

Descomprime el archivo

Opción 2 (Avanzado)
git clone https://github.com/TITOPE773/moville.git
 Cómo ejecutar el proyecto
Requisitos:

XAMPP, WAMP o cualquier servidor local con PHP

Pasos:

Copia la carpeta del proyecto en:

C:\xampp\htdocs\

Inicia Apache en XAMPP

Abre tu navegador y entra a:

https://localhost/movile/frontend/index.html
